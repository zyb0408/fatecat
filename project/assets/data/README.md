# FateCat 静态数据资产

`assets/data/` 是项目的静态数据资产层。

## 分区

| 路径 | 用途 | 是否默认运行依赖 |
| --- | --- | --- |
| `china_coordinates.csv` | 中国行政区经纬度数据 | 是 |
| `classics/*.txt` | 已整理命理古籍与基础语料 | 可选 |
| `classics/raw/` | 本地原始书籍、PDF、讲义、OCR 文本 | 否 |
| `calendar/solar_terms/raw/` | 本地交节时间原始表格 | 否 |

## 原始资料规则

- `raw/` 只作为本地来源资料和人工复核材料。
- `raw/` 已被 `.gitignore` 和导出脚本排除，避免大文件、版权不明资料或未清洗 OCR 进入发布包。
- 后续要用于生产逻辑时，必须先完成来源复核、结构化抽取、字段契约和回归测试。
