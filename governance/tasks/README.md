# Governance Tasks

这里放项目任务包、任务树、执行波次、closeout 和任务级候选 lessons。长期有效规则应拆分后晋升到 `evidence/`、`architecture-gates/`、`processes/` 或 `standards/`。

- `lessons.md`：任务级候选教训整理池。
- `INDEX.md`：active 任务包索引，只登记 `<task-id>-<slug>/` 目录。
- `<task-id>-<slug>/`：当前任务包；目录名必须满足 auto-tasks 批量校验。

历史任务包归档在 `governance/archive/tasks/project-history/`，不放在 active tasks 根。
