# 零美化语义界面标准

## 适用范围

本标准适用于 FateCat 的 HTML 优先页面，尤其是：

- `GET /web`
- 工程报表页
- 状态页
- 调试页
- 本地工具页
- 服务端直出结果页

这些页面不是营销页，也不是视觉型产品页面；它们的目标是人工可读、机器可解析、可复制、可测试、可审计。

## 单一真相源

- 设计规范：`/home/lenovo/.codex/Design.md`
- Web 实现：`domains/experience-delivery/services/fatecat-delivery/src/web_ui.py`
- 回归门禁：`tests/regression/test_web_html.py`
- 架构 gate：`governance/architecture-gates/rules/GATE-0001-Web-HTML-禁止自定义前端样式.md`

## 强制要求

- 页面通常用 `h1` 表达主题；`/web` 黄金生产空间例外允许只保留 `<title>`，避免三窗口之外出现第四个视觉块。
- 稳定区块用 `h2/h3` 表达层级。
- 元信息用 `dl/dt/dd`。
- 导航和链接用真实 `a` 与 `nav/ul/li`。
- 输入和查询用原生 `form/fieldset/legend/label/input/select/button`。
- 结构化数据用 `tabulate(tablefmt="psql")` 输出到 `<pre><code>`。
- Markdown、JSON、日志、命令、错误和机器可读片段直接放进 `<pre><code>`。
- 长但非核心内容可用 `details/summary`。
- 核心内容必须由服务端直接写入页面，无 JavaScript 也可读。

## 授权例外

用户在 2026-06-15 明确要求 `/web` 借鉴 `D:\.projects\pdf` 工作台的黄金分割三块生产空间：左上 TradeCat Labs 资产声明、右上服务端生成报告、底部参数输入。该布局是操作结构，不是普通文档流。

允许范围仅限：

- 一个 `<style>` 块定义黄金比例变量：`61.8034%` / `38.1966%` / `1.61803398875fr` / `1fr`。
- 一个原生 GET `<form class="web-production-grid">` 作为三块生产空间容器。
- 三个语义 `<section>`：`web-production-brand`、`web-production-report`、`web-production-input`，以及为了复现 pdf 工作台标题行和控件行而存在的 `web-production-report-header`、`web-production-submit-button`、`web-production-control-grid`。
- CSS 必须对齐 pdf 工作台外壳：全屏 `html/body`、深色工作台背景、面板边界、黄金比例 `grid-template-*`、`gap: 0`、三块 `grid-column/grid-row`、滚动行为、原生控件可读性和窄屏自然堆叠。

该例外不得扩大为圆角、阴影、卡片、动画、图标、营销视觉或复杂前端组件的许可；颜色、字体、背景和边框只允许作为 pdf 工作台全屏外壳与原生控件可读性的直接复刻。

## 禁止事项

- 除上述黄金三块生产空间例外外，禁止 `<style>`、外部 CSS、行内 `style=`。
- 除 pdf 工作台全屏外壳例外外，禁止自定义颜色、背景、字体、字号、间距、边框、圆角、阴影。
- 除上述窄屏自然堆叠例外外，禁止 `@media` 和响应式布局优化。
- 除上述 `web-production-*` 结构 class 外，禁止视觉 class、卡片、装饰性容器、图标装饰和动画。
- 禁止为了视觉效果隐藏字段或折叠核心数据。
- 禁止把普通链接伪装成按钮。
- 禁止把 psql ASCII 表格、Markdown 或 JSON 改造成不可复制组件。

## 允许的增强

JavaScript 只能作为渐进增强，例如复制 Markdown。增强必须满足：

- 不负责核心内容可见性。
- 不做视觉美化。
- 失败时页面仍可手动复制和提交。

```bash
python -m pytest -q tests/regression/test_web_html.py

```

## 审查提示

如果一次 Web 变更声称“改善体验”但引入黄金三块生产空间之外的 CSS、视觉 class、卡片或响应式布局，默认判定为违反本标准。先删除视觉实现，再讨论是否存在真实的信息结构或操作结构问题。
