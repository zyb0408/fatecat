### 2.5 今日吉凶时 `luckyHours`

**数据源**: lunar-python

**实现方式**:
```python
def _calc_lucky_hours(self) -> Dict[str, Any]:
    """计算今日吉凶时辰"""
    from lunar_python import Solar
    from utils.timezone import now_cn
    
    today = now_cn()
    solar = Solar.fromDate(today)
    lunar = solar.getLunar()
    
    lucky = []
    unlucky = []
    
    # 获取时辰吉凶
    times = lunar.getTimes()
    for t in times:
        hour_info = {
            "name": t.getZhi() + "时",
            "range": t.getMinHm() + "-" + t.getMaxHm(),
            "ganzhi": t.getGanZhi()
        }
        if t.getYi():  # 有宜则为吉时
            lucky.append(hour_info)
        else:
            unlucky.append(hour_info)
    
    return {
        "lucky": lucky,
        "unlucky": unlucky,
        "currentHour": self._get_current_hour_info(lunar)
    }
```

**输出字段**: `luckyHours`
