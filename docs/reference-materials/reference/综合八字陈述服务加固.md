# 综合八字陈述服务加固

> 日期：2026-05-07
> 范围：默认综合八字报告、纯分析 JSON、节气边界、证据字段、隐私与验收门禁。

## 当前生产边界

- 默认报告体系只输出 `bazi`：综合八字主线 + 袁天罡称骨民俗附录。
- 紫微斗数只通过 `reportSystem=ziwei` 独立输出。
- 黄历/择日已作为 `almanac` 生产 capability 独立提供；建除、六爻、梅花、奇门、大六壬、风水九星、姓名合婚等仍不得混入综合八字默认报告。
- Web/API/Bot/Markdown 展示层回显用户提交的出生地区；经纬度仍用于地区解析、真太阳时和后端计算。

## 证据字段

`analysisEvidence` 是 JSON/API 审计字段，默认隐藏，不进入 Markdown 正文。当前覆盖：

- `dayMaster`：日主、月令、五行强弱依据。
- `wuxingPreference`：调候与五行喜忌依据。
- `pattern`：格局与月令、透干、藏干依据。
- `ganzhiRelations`：四柱干支关系依据。
- `spirits`：神煞辅助依据，权重为 `auxiliary`。
- `boneWeight`：袁天罡称骨民俗附录，权重为 `folk`。
